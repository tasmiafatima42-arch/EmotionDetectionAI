# Emotion Detection AI System

## Overview
A Flask and OpenCV based Emotion Detection System that detects emotions through webcam and displays emoji-based feedback.

## Features
- User Login & Registration
- Face Detection
- Emotion Detection
- Start/Stop Camera
- Emotion History
- Graph Visualization
- Modern Dashboard UI

## Technologies Used
- Python
- Flask
- OpenCV
- Matplotlib
- HTML/CSS

## Installation

```bash
pip install -r requirements.txt
python app.py
